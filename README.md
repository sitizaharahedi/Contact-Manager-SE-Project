# Contact Manager - Portfolio Project

A simple **Full-Stack Contact Manager** demonstrating backend and frontend skills.  
This project is for **portfolio purposes only** to showcase my work in Python, Flask, and RESTful API development.

---

## Project Overview

- **Frontend:** React (Vite) for a responsive UI  
- **Backend:** Flask REST API using SQLAlchemy ORM  
- **Database:** SQLite (persistent storage for contacts)  
- **Functionality:**  
  - Create, Read, Update, Delete (CRUD) contacts  
  - Modal form for creating/updating contacts  
  - RESTful API endpoints for backend interaction  

This project highlights my ability to build **REST APIs, connect to a database, and integrate with a frontend**.

---

## Tech Stack

**Backend:** Python, Flask, Flask-CORS, Flask-SQLAlchemy, SQLite  
**Frontend:** React 18, Vite, JSX, CSS  

---

## API Endpoints

| Method | Endpoint                  | Description                |
|--------|---------------------------|----------------------------|
| GET    | /contacts                 | Retrieve all contacts      |
| POST   | /create_contact           | Create a new contact       |
| PATCH  | /update_contact/<id>      | Update contact by ID       |
| DELETE | /delete_contact/<id>      | Delete contact by ID       |
